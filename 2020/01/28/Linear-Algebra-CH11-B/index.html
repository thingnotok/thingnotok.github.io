LA Lec11(B): Orthogonality II

Projection

  • Example:
    把$b$投影到$a$
    $$b=\begin{bmatrix}1\1\1\end{bmatrix}, a=\begin{bmatrix}1\2\2\end{bmatrix}$$
    Projection matrix
    $$P=\frac{aa^T}{a^Ta}=\frac{1}{9}\begin{bmatrix}1\2\2\end{bmatrix}\begin{bmatrix}1\ 2\ 2\end{bmatrix}$$
    $$P=\frac{1}{9}\begin{bmatrix}1\ 2\ 2\2\ 4\ 4\2\ 4\ 4\end{bmatrix}$$
    Projection vector
    $$p=Pb=\frac{1}{9}\begin{bmatrix}1\ 2\ 2\2\ 4\ 4\2\ 4\ 4\end{bmatrix}\begin{bmatrix}1\1\1\end{bmatrix}$$
    $$p=\frac{1}{9}\begin{bmatrix}5\10\10\end{bmatrix}$$
    Verify: 算出error vector 做驗算($e\perp p$ or $e\perp a$)
    $$e=b-p=\begin{bmatrix}1\1\1\end{bmatrix}-\frac{1}{9}\begin{bmatrix}5\10\10\end{bmatrix}=\frac{1}{9}\begin{bmatrix}4\-1\-1\end{bmatrix}$$
    $$a^Te=\frac{1}{9}\begin{bmatrix}1\ 2\ 2\end{bmatrix}\begin{bmatrix}4\-1\-1\end{bmatrix}=0$$

  • Note that $P^2=P$:
    $$P^2=PP=\frac{1}{9}\begin{bmatrix}1\ 2\ 2\2\ 4\ 4\2\ 4\ 4\end{bmatrix}\frac{1}{9}\begin{bmatrix}1\ 2\ 2\2\ 4\ 4\2\ 4\ 4\end{bmatrix}$$
    $$=\frac{1}{81}\begin{bmatrix}9\ 18\ 18\18\ 36\ 36\18\ 36\ 36\end{bmatrix}=P$$
    (Since projecting a second time does not change anything)
    (因為任何向量投影在a上，都會是a的倍數(跟a平行)，再投影一次還是在a上，所以投影兩次的結果等同於投影一次)
    In general,
    $$P^2=\frac{aa^T}{a^Ta}\frac{aa^T}{a^Ta}=\frac{a(a^Ta)a^T}{(a^Ta)^2}=\frac{a^Ta}{(a^Ta)^2}aa^T=\frac{aa^T}{a^Ta}=P$$
    ($aa^T$可以被提出來跟消掉，是因為$a^Ta$是個實數)

Projection onto a Subspace

Start with $n$ independent vectors $a_1, a_2, …,a_n in R^m$
這邊假設有個 $R^m$ 的空間，並包含n個獨立向量($n \leq m$)，然後想把一個向量投影到這個空間上
Ｗe want to find
$p=\hat{x_1}a_1+\hat{x_2}a_2+ … +\hat{x_n}a_n$
as a projection of a given vector $b$
我們想要找到一個向量p使得，b投影到subspace上後等於p，因為p在subspace上，所以可以寫成$a_1, a_2, …,a_n$的線性組合
Let $A = \begin{bmatrix}a_1\ a_2\ …\ a_n\end{bmatrix}_{m\times n}$
Then $p=\begin{bmatrix}a_1\ a_2\ …\ a_n\end{bmatrix}\begin{bmatrix}\hat{x_1}\\hat{x_2}\.\.\\hat{x_n}\end{bmatrix}=A\hat{x}$
因為我們前面有提到$p$是$a_1, a_2, …,a_n$的線性組合，所以$p$在$A$的column space裡面，所以可以寫成像上面這樣
這邊可以跟之前投影到line上一樣去想，本來投影到line上之後error vector $e$($e=b-p$)就是$b$投影到向量$a$上後，夾角對出去那條垂直的線，所以$e$會跟$a$正交($e\perp a$)。那現在$e$就是b投影到平面A上面的後，夾角對出去的那條垂直線，在這個情況下，$e$會跟平面A上的所有向量正交，所以我們這邊後面要看的就是$e\perp (a_1, a_2, … a_n)$。
The error vector $b-A\hat{x}$ should be orthogonal to the subspace spanned by $a_1, a_2, … a_n$
這邊這個被spanned出來的subspace 其實就是$A$的column space
$$a_1^T(b-A\hat{x})=0\a_2^T(b-A\hat{x})=0\.\.\a_n^T(b-A\hat{x})=0$$
=> $\begin{bmatrix}a_1^T\a_2^T\.\.\a_n^T\end{bmatrix}\begin{bmatrix}b-A\hat{x}\end{bmatrix}=\begin{bmatrix}0\0\.\.\0\end{bmatrix}$
=> $A^T(b-A\hat{x})=0$
這邊很特別$e$要跟column space orthogonal其實就是代表e要在left nullspace裡面，就是$A^T$的null space
這一段的關鍵就是，我們在定義error vector $e$後，我們就知道error vector要跟每一個獨立向量$a_n$ orthogonal 所以就會寫出像上面那樣的式子
=> $A^TA\hat{x}=A^Tb$(從上面往下推)
這邊$A^TA$是$n\times n$的正方形矩陣， $\hat{x}$是$n\times 1$, $A^Tb$是$n\times 1$。就是在解一個$n\times n$的聯立方程式
如果我們想要往後探討，這邊稍微調整一下上面那一式的寫法
The matrix $A^TA$ is square symmetric(他取轉置transpose之後還是他自己)
and it is invertible iff $a_1, a_2, … a_n$ are linear independent(這邊之後會證明，現在先相信啟超)
=> $\hat{x}=(A^TA)^{-1}A^Tb$
$\therefore p=A\hat{x}=A(A^TA)^{-1}A^Tb$ (projection vector 可以寫成這樣)
所以Projection matrix就是 $A(A^TA)^{-1}A^T$

  • Summary
    投影到平面與 n=1(投影到line上面)做比較: 這邊剛好就是$A$變成只有一個column之後的形式，
    可以看到有些$(A^TA)^{-1}$在n=1的時候被寫成$\frac{1}{a^Ta}$
    這是因為n=1時這一項剛好會是個實數，所以可以寫成除的形式

    • 係數組合 $\hat{x}=(A^TA)^{-1}A^Tb$

      • $\hat{x}=\frac{a^Tb}{a^Ta}$

    • 投影向量 $p=A\hat{x}=A(A^TA)^{-1}A^Tb$

      • $p=\frac{a^Tb}{a^Ta}a$

    • 投影矩陣$P=A(A^TA)^{-1}A^T$

      • $P=\frac{a^Ta}{a^Ta}$

  • Note $P^2 = P$
    $$P^2=A(A^TA)^{-1}(A^TA)(A^TA)^{-1}A^T\=A(A^TA)^{-1}A^T=P$$
    同理，投影兩次結果還是跟投影一次一樣

  • Example:把$b$投影到$A$上
    $A=\begin{bmatrix}1\ 0\1\ 1\1\ 2\end{bmatrix}$ and $b=\begin{bmatrix}6\0\0\end{bmatrix}$
    Projeciton vector:
    $$A^TA=\begin{bmatrix}1\ 1\ 1\0\ 1\ 2\end{bmatrix}\begin{bmatrix}1\ 0\1\ 1\1\ 2\end{bmatrix}=\begin{bmatrix}3\ 3\3\ 5\end{bmatrix}$$
    $$A^Tb=\begin{bmatrix}1\ 1\ 1\0\ 1\ 2\end{bmatrix}\begin{bmatrix}6\0\0\end{bmatrix}=\begin{bmatrix}6\0\end{bmatrix}$$
    $$A^TA\hat{x}=A^Tb$$
    => $\begin{bmatrix}3\ 3\3\ 5\end{bmatrix}\begin{bmatrix}\hat{x_1}\\hat{x_2}\end{bmatrix}=\begin{bmatrix}6\0\end{bmatrix}$
    => $\begin{bmatrix}\hat{x_1}\\hat{x_2}\end{bmatrix}=\begin{bmatrix}5\-3\end{bmatrix}$
    $\therefore p=A\hat{x}=\begin{bmatrix}1\ 0\1\ 1\1\ 2\end{bmatrix}\begin{bmatrix}5\-3\end{bmatrix}=\begin{bmatrix}5\2\-1\end{bmatrix}$
    Verify:
    $e=b-p=\begin{bmatrix}6\0\0\end{bmatrix}-\begin{bmatrix}5\2\-1\end{bmatrix}=\begin{bmatrix}1\-2\1\end{bmatrix}$
    $A^Te=\begin{bmatrix}1\ 1\ 1\0\ 1\ 2\end{bmatrix}\begin{bmatrix}1\-2\1\end{bmatrix}=\begin{bmatrix}0\0\end{bmatrix}$
    $e$ in left nullspace of $A$
    Projection Matrix:
    $P=A(A^TA)^{-1}A^T=\frac{1}{6}\begin{bmatrix}5\ 2\ -1\2\ 2\ 2\-1\ 2\ 5\end{bmatrix}$
    $P^2=P$

  • Chaim: $rank(A^TA)=rank(AA^T)=rank(A)$
    只要能夠證明這幾個矩陣的rank數目相同，就能夠證明前面的inverterable
    其中$A^TA_{n\times n}$, $AA^T_{m\times m}$, $A_{m\times n}$都是不同形狀的矩陣

    • Proof:
      要證明兩個rank一樣，只要證明兩個null space dimension一樣。(因為null space的dimension = n - r)
      We first show that $A^TA$ has the same nullspace as A
      i.e., $A^TAx=0 <=> Ax=0$
      “<=”
      $Ax=0$
      => $A^TA=A^T0=0$
      => $A^TAx=0$
      “=>”
      $A^TAx=0$
      => $x^TA^TAx=x^T0$
      左右兩邊同乘$x^T$湊成對稱的形式，會發現前面就是$x^TA^T$就是$(Ax)^T$
      => $(Ax)^T(Ax)=0$
      => $||Ax||^2=0$
      因為向量長度為零的向量就是零向量
      => Ax=0
      Therefore, $N(A^TA)=N(A)$,
      which implies $n-rank(A^TA)=b-rank(A)$
      => $rank(A^TA)=rank(A)$
      同理我們只要把$A^T$換成$A$就能得到一樣的結果
      Similarly, putting $A^T$ as $A$, we can get $rank(AA^T)=rank(A^T)=rank(A)$
