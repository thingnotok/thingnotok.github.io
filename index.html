LA Lec12(A): Orthogonality III

  • Chaim: $rank(A^TA)=rank(AA^T)=rank(A)$

  • Remark 1: $A^TA$ is invertible iff $A$ has independent columns

    • Proof:
      A has independent columns
      有獨立的columns的話，代表這些columns就是A的basis，有幾個column就代表A的rank有多少
      <=> $rank(A) = n$
      <=> $rank(A^TA)=n$
      <=> $A^TA$ is invertible

  • Remark 2: When A has independent columns, $A^TA$ is square, symmetric, and invertible
    這邊square跟symmetric不管有沒有independent都成立

Leasr Square Approximations

It often happens that $Ax=b$ has no solution.
=> We cannot always get the error $e=b-Ax$ down to zero.
In this case, we may want the length of $e$, or $||e||^2$ as small as possible.
=> least squares solution
有時候不是所有$Ax=b$都有解，可能方程式太多，變數太少，解不出來，這時候我們可能會想要找一組變數，使得這些方程式有最接近的解。最接近就代表error vector越小越好，或者說這個vector的長度越短越好，這種問題稱作最小二乘。

  • Example:
    Find the closet line to the points (0, 6), (1, 0), and (2, 0)
    二維空間上有三點，我們想找一條線，使得這條線離這三點最近
    令直線方程式為 $b=C+Dt$ 這邊b是縱軸，t是橫軸
    把這三點條件寫出來可以得到
    $C+D0=6$
    $C+D    1=0$
    $C+D*2=0$
    => $A=\begin{bmatrix}1\ 0\1\ 1\1\ 2\end{bmatrix}, x=\begin{bmatrix}C\D\end{bmatrix}, b=\begin{bmatrix}6\0\0\end{bmatrix}$
    $Ax=b$ is not solvable.
    We want to make $||b-Ax||^2$ as small as possible.
    How?
    這邊error 的意思是點到線之間的距離，可以參考影片17:10的地方
    這邊一般的用途像是，我們做實驗觀察到三個點，我們可能預期結果是一條直線，那我們就可以透過這三個數據點去預測出一條直線。
    這邊的問題沒有解，也等同於$b$不在$A$的column space裡面，那我們想找一個vector與$b$的距離最短，就可以把$b$投影到A的column space裡面就能夠得到那個使error最短的vector。這邊建議看老師畫圖(22:15)
    By geometry, the best fit is $\hat{x}$ such that $p=A\hat{x}$ is the projection of $b$ onto $C(A)$
    $A^T(b-A\hat{x})=0$
    => $A^TA\hat{x}=A^Tb$
    $\begin{bmatrix}3\ 3\3\ 5\end{bmatrix}\begin{bmatrix}C\D\end{bmatrix} = \begin{bmatrix}6\0\end{bmatrix}$
    $\hat{x}=\begin{bmatrix}C\D\end{bmatrix}=\begin{bmatrix}5\-3\end{bmatrix}$
    Therefore the best line for the 3 points is $b=5-3t$.

    By linear algebra,
    recall $x=x_r+x_n$ where $x_r\in C(A^T)$, and $x_n\in N(A)$
    任何$R^n$的向量可以拆成一組row space vector 及一組null space vector的組合
    $b=p+e$
    where $p^Te=0$
    $b\in R^m$, $p \in C(A)$, $e\in N(A^T)$
    因為$b$在$R^m$不是在$R^n$ 所以整個一起取transpose，得到相似的結論
    We know that
    $A\hat{x}=b=p+e$
    is impossible
    and $A\hat{x}=p$ is solvable
    $||Ax-b||^2 = ||Ax-p-e||^2$
    $=(Ax-p-e)^T(Ax-p-e)$
    $=||Ax-p||^2+||e||^2-(Ax-p)^Te-e^T(Ax-p)$
    (其中$(Ax-p)^Te=0$, 因為$(Ax-p)\in C(A)$ and $e\in N(A^T)$)
    (前面有提到過$C(A)$跟$N(A^T)$是orthogonal complement所以內積為零)
    $=||Ax-p||^2+||e||^2\geq ||e||^2$
    $\therefore ||Ax-p||^2$ is minimized when $x=\hat{x}$.
    i.e., $\min_{x}||Ax-b||^2=||A\hat{x}-b||^2$
    $||A\hat{x}-p||^2+||e||^2=||e||^2$,
    where $A^TA\hat{x}=A^Tb$ and $p=A\hat{x}$

LA Lec11(B): Orthogonality II

Projection

  • Example:
    把$b$投影到$a$
    $$b=\begin{bmatrix}1\1\1\end{bmatrix}, a=\begin{bmatrix}1\2\2\end{bmatrix}$$
    Projection matrix
    $$P=\frac{aa^T}{a^Ta}=\frac{1}{9}\begin{bmatrix}1\2\2\end{bmatrix}\begin{bmatrix}1\ 2\ 2\end{bmatrix}$$
    $$P=\frac{1}{9}\begin{bmatrix}1\ 2\ 2\2\ 4\ 4\2\ 4\ 4\end{bmatrix}$$
    Projection vector
    $$p=Pb=\frac{1}{9}\begin{bmatrix}1\ 2\ 2\2\ 4\ 4\2\ 4\ 4\end{bmatrix}\begin{bmatrix}1\1\1\end{bmatrix}$$
    $$p=\frac{1}{9}\begin{bmatrix}5\10\10\end{bmatrix}$$
    Verify: 算出error vector 做驗算($e\perp p$ or $e\perp a$)
    $$e=b-p=\begin{bmatrix}1\1\1\end{bmatrix}-\frac{1}{9}\begin{bmatrix}5\10\10\end{bmatrix}=\frac{1}{9}\begin{bmatrix}4\-1\-1\end{bmatrix}$$
    $$a^Te=\frac{1}{9}\begin{bmatrix}1\ 2\ 2\end{bmatrix}\begin{bmatrix}4\-1\-1\end{bmatrix}=0$$

  • Note that $P^2=P$:
    $$P^2=PP=\frac{1}{9}\begin{bmatrix}1\ 2\ 2\2\ 4\ 4\2\ 4\ 4\end{bmatrix}\frac{1}{9}\begin{bmatrix}1\ 2\ 2\2\ 4\ 4\2\ 4\ 4\end{bmatrix}$$
    $$=\frac{1}{81}\begin{bmatrix}9\ 18\ 18\18\ 36\ 36\18\ 36\ 36\end{bmatrix}=P$$
    (Since projecting a second time does not change anything)
    (因為任何向量投影在a上，都會是a的倍數(跟a平行)，再投影一次還是在a上，所以投影兩次的結果等同於投影一次)
    In general,
    $$P^2=\frac{aa^T}{a^Ta}\frac{aa^T}{a^Ta}=\frac{a(a^Ta)a^T}{(a^Ta)^2}=\frac{a^Ta}{(a^Ta)^2}aa^T=\frac{aa^T}{a^Ta}=P$$
    ($aa^T$可以被提出來跟消掉，是因為$a^Ta$是個實數)

Projection onto a Subspace

Start with $n$ independent vectors $a_1, a_2, …,a_n in R^m$
這邊假設有個 $R^m$ 的空間，並包含n個獨立向量($n \leq m$)，然後想把一個向量投影到這個空間上
Ｗe want to find
$p=\hat{x_1}a_1+\hat{x_2}a_2+ … +\hat{x_n}a_n$
as a projection of a given vector $b$
我們想要找到一個向量p使得，b投影到subspace上後等於p，因為p在subspace上，所以可以寫成$a_1, a_2, …,a_n$的線性組合
Let $A = \begin{bmatrix}a_1\ a_2\ …\ a_n\end{bmatrix}_{m\times n}$
Then $p=\begin{bmatrix}a_1\ a_2\ …\ a_n\end{bmatrix}\begin{bmatrix}\hat{x_1}\\hat{x_2}\.\.\\hat{x_n}\end{bmatrix}=A\hat{x}$
因為我們前面有提到$p$是$a_1, a_2, …,a_n$的線性組合，所以$p$在$A$的column space裡面，所以可以寫成像上面這樣
這邊可以跟之前投影到line上一樣去想，本來投影到line上之後error vector $e$($e=b-p$)就是$b$投影到向量$a$上後，夾角對出去那條垂直的線，所以$e$會跟$a$正交($e\perp a$)。那現在$e$就是b投影到平面A上面的後，夾角對出去的那條垂直線，在這個情況下，$e$會跟平面A上的所有向量正交，所以我們這邊後面要看的就是$e\perp (a_1, a_2, … a_n)$。
The error vector $b-A\hat{x}$ should be orthogonal to the subspace spanned by $a_1, a_2, … a_n$
這邊這個被spanned出來的subspace 其實就是$A$的column space
$$a_1^T(b-A\hat{x})=0\a_2^T(b-A\hat{x})=0\.\.\a_n^T(b-A\hat{x})=0$$
=> $\begin{bmatrix}a_1^T\a_2^T\.\.\a_n^T\end{bmatrix}\begin{bmatrix}b-A\hat{x}\end{bmatrix}=\begin{bmatrix}0\0\.\.\0\end{bmatrix}$
=> $A^T(b-A\hat{x})=0$
這邊很特別$e$要跟column space orthogonal其實就是代表e要在left nullspace裡面，就是$A^T$的null space
這一段的關鍵就是，我們在定義error vector $e$後，我們就知道error vector要跟每一個獨立向量$a_n$ orthogonal 所以就會寫出像上面那樣的式子
=> $A^TA\hat{x}=A^Tb$(從上面往下推)
這邊$A^TA$是$n\times n$的正方形矩陣， $\hat{x}$是$n\times 1$, $A^Tb$是$n\times 1$。就是在解一個$n\times n$的聯立方程式
如果我們想要往後探討，這邊稍微調整一下上面那一式的寫法
The matrix $A^TA$ is square symmetric(他取轉置transpose之後還是他自己)
and it is invertible iff $a_1, a_2, … a_n$ are linear independent(這邊之後會證明，現在先相信啟超)
=> $\hat{x}=(A^TA)^{-1}A^Tb$
$\therefore p=A\hat{x}=A(A^TA)^{-1}A^Tb$ (projection vector 可以寫成這樣)
所以Projection matrix就是 $A(A^TA)^{-1}A^T$

  • Summary
    投影到平面與 n=1(投影到line上面)做比較: 這邊剛好就是$A$變成只有一個column之後的形式，
    可以看到有些$(A^TA)^{-1}$在n=1的時候被寫成$\frac{1}{a^Ta}$
    這是因為n=1時這一項剛好會是個實數，所以可以寫成除的形式

    • 係數組合 $\hat{x}=(A^TA)^{-1}A^Tb$

      • $\hat{x}=\frac{a^Tb}{a^Ta}$

    • 投影向量 $p=A\hat{x}=A(A^TA)^{-1}A^Tb$

      • $p=\frac{a^Tb}{a^Ta}a$

    • 投影矩陣$P=A(A^TA)^{-1}A^T$

      • $P=\frac{a^Ta}{a^Ta}$

  • Note $P^2 = P$
    $$P^2=A(A^TA)^{-1}(A^TA)(A^TA)^{-1}A^T\=A(A^TA)^{-1}A^T=P$$
    同理，投影兩次結果還是跟投影一次一樣

  • Example:把$b$投影到$A$上
    $A=\begin{bmatrix}1\ 0\1\ 1\1\ 2\end{bmatrix}$ and $b=\begin{bmatrix}6\0\0\end{bmatrix}$
    Projeciton vector:
    $$A^TA=\begin{bmatrix}1\ 1\ 1\0\ 1\ 2\end{bmatrix}\begin{bmatrix}1\ 0\1\ 1\1\ 2\end{bmatrix}=\begin{bmatrix}3\ 3\3\ 5\end{bmatrix}$$
    $$A^Tb=\begin{bmatrix}1\ 1\ 1\0\ 1\ 2\end{bmatrix}\begin{bmatrix}6\0\0\end{bmatrix}=\begin{bmatrix}6\0\end{bmatrix}$$
    $$A^TA\hat{x}=A^Tb$$
    => $\begin{bmatrix}3\ 3\3\ 5\end{bmatrix}\begin{bmatrix}\hat{x_1}\\hat{x_2}\end{bmatrix}=\begin{bmatrix}6\0\end{bmatrix}$
    => $\begin{bmatrix}\hat{x_1}\\hat{x_2}\end{bmatrix}=\begin{bmatrix}5\-3\end{bmatrix}$
    $\therefore p=A\hat{x}=\begin{bmatrix}1\ 0\1\ 1\1\ 2\end{bmatrix}\begin{bmatrix}5\-3\end{bmatrix}=\begin{bmatrix}5\2\-1\end{bmatrix}$
    Verify:
    $e=b-p=\begin{bmatrix}6\0\0\end{bmatrix}-\begin{bmatrix}5\2\-1\end{bmatrix}=\begin{bmatrix}1\-2\1\end{bmatrix}$
    $A^Te=\begin{bmatrix}1\ 1\ 1\0\ 1\ 2\end{bmatrix}\begin{bmatrix}1\-2\1\end{bmatrix}=\begin{bmatrix}0\0\end{bmatrix}$
    $e$ in left nullspace of $A$
    Projection Matrix:
    $P=A(A^TA)^{-1}A^T=\frac{1}{6}\begin{bmatrix}5\ 2\ -1\2\ 2\ 2\-1\ 2\ 5\end{bmatrix}$
    $P^2=P$

  • Chaim: $rank(A^TA)=rank(AA^T)=rank(A)$
    只要能夠證明這幾個矩陣的rank數目相同，就能夠證明前面的inverterable
    其中$A^TA_{n\times n}$, $AA^T_{m\times m}$, $A_{m\times n}$都是不同形狀的矩陣

    • Proof:
      要證明兩個rank一樣，只要證明兩個null space dimension一樣。(因為null space的dimension = n - r)
      We first show that $A^TA$ has the same nullspace as A
      i.e., $A^TAx=0 <=> Ax=0$
      “<=”
      $Ax=0$
      => $A^TA=A^T0=0$
      => $A^TAx=0$
      “=>”
      $A^TAx=0$
      => $x^TA^TAx=x^T0$
      左右兩邊同乘$x^T$湊成對稱的形式，會發現前面就是$x^TA^T$就是$(Ax)^T$
      => $(Ax)^T(Ax)=0$
      => $||Ax||^2=0$
      因為向量長度為零的向量就是零向量
      => Ax=0
      Therefore, $N(A^TA)=N(A)$,
      which implies $n-rank(A^TA)=b-rank(A)$
      => $rank(A^TA)=rank(A)$
      同理我們只要把$A^T$換成$A$就能得到一樣的結果
      Similarly, putting $A^T$ as $A$, we can get $rank(AA^T)=rank(A^T)=rank(A)$

LA Lec11(A): Orthogonality II

Property 1

  • Claim:
    Every b in the column space comes from one and only one vector in the row space.
  • Proof:
    b 一定存在，這邊只證明唯一
    Let $b=Ax_r=Ax_r’ \in C(A)$ where $x_r, x_r’ \in C(A^T)$
    => $A(x_r-x_r’)=0$
    => $(x_r-x_r’) \in N(A)$
    Since also $x_r-x_r’ \in C(A^T)$,
    we can have $x_r-x_r’=0$,
    yeilding $x_r=x_r’$
  • Example:
    $A=\begin{bmatrix}1\ 2\3\ 6\end{bmatrix}$
    We can have $x=\begin{bmatrix}4\3\end{bmatrix}=\begin{bmatrix}2\4\end{bmatrix}_{x_r}+\begin{bmatrix}2\-1\end{bmatrix}_{x_n}$
    找的方法先在兩個space中個找一組basis，之後把向量對basis做展開即可
    或是可以把A投影到row 或是null space，就能得到其中一解

Def: Projection

  • Example:
    $$b=\begin{bmatrix}
    2\\3\\4
    \end{bmatrix}$$
    what are the projections of $b$ on the z axis and the xy plane
    投影到z軸的結果是$p_1=\begin{bmatrix}
    0\\0\\4
    \end{bmatrix}$，投影到xy plane的結果是$p_2=\begin{bmatrix}
    2\\3\\0
    \end{bmatrix}$的向量
    把投影想成一個矩陣乘法
    $p_1=\begin{bmatrix}0\ 0\ 0\0\ 0\ 0\0\ 0\ 1\end{bmatrix}\begin{bmatrix}2\3\4\end{bmatrix}=\begin{bmatrix}0\0\4\end{bmatrix}$
    $p_2=\begin{bmatrix}1\ 0\ 0\0\ 1\ 0\0\ 0\ 0\end{bmatrix}\begin{bmatrix}2\3\4\end{bmatrix}=\begin{bmatrix}2\3\0\end{bmatrix}$
    可以發現 $p_1+p_2=b$ 原因是z軸跟xy plane互為orthogonal subspace
    因為兩個相加等於b 也可以發現 $p_1+p_2=I$

Def: Projection onto a Line

Ｗe want to find the projection of $b=\begin{bmatrix}b1\ b2\.\.\bm\end{bmatrix}$ onto the line in the direction of $a=\begin{bmatrix}a1\ a2\.\.\am\end{bmatrix}$
Let $p=\hat{x}a$, then $e=b-p=b-\hat{x}a$, and $e\perp a$
=> $a^Te=a^T(b-\hat{x}a)=0$
=> $a^Tb-\hat{x}a^Ta=0$
=> $\hat{x}=\frac{a^Tb}{a^Ta}$
$\therefore p = \hat{x}a=\frac{a^Tb}{a^Ta}a$
If $b=a$, then $\hat{x}=1$, $p=a$
If $b$ and $a$ are orthogonal, then $\hat{x}=0$ and $p=0$

接下來從投影向量$p$來計算投影矩陣$P$
$p=a\hat{x}=a\frac{a^Tb}{a^Ta}=\frac{aa^T}{a^Ta}b=Pb$
where $P=\frac{aa^T}{a^Ta}$ is the projection martix
其中 $aa^T$是 $m\times m$ 的矩陣， $a^Ta$是實數(向量長度平方)

過程中的 $e=b-p=b-Pb=(I-P)b$

LA Lec10(B): Orthogonality I

Summary

  • $(C(A^T))^\perp = N(A)$
  • $(N(A))^\perp = C(A^T)$
  • $(C(A))^\perp = N(A^T)$
  • $(N(A^T))^\perp = C(A)$

Fundamental Theorem of Linear Algebra Part II

  • $N(A)$ is the orthogonal complement of $C(A^T)$ (in $R^n$)
  • $N(A^T)$ is the orthogonal complemtn of $C(A)$ (in $R^m$)

Propertey 1

  • Claim:
    $v_1, v_2, v_3, …, v_n$ are independent in $R^n$ and only if $v_1, v_2, v_3,…, v_n$ span $R^n$
  • Proof:
    Let $A_{n \times n}=[v_1, v_2, v_3, …, v_n]$
    • “=>”
      If $v_1, v_2, … v_n$ are independent (and they form a basis for $C(A)$)
      then $rank(A) = n$
      Hence $A$ is inverterble.
      And $Ax=b$ is solvable for every $b \in R^n$. Then $C(A)=R^n$.
      Therefore, $v_1, v_2, …, v_n$ span $R^n$
    • “<=”
      If $v_1, v_2, … v_n$ span $R^n$ then $Ax=b$ is solvable for every $b\ in R^n$.
      Hence elimination produces no zero rows.
      There are n pivots and thus $rank(A)=n$. The nullspace $N(A)={0}$, which inplies $v_1, v_2, …, v_n$ are independent.
  • Remark:
    Any n independent vectors form a basis in $R^n$. Also any vectors that span $R^n$ are basis for $R^n$.

Property 2

  • Claim:
    If $V\perp W$, then $V\cap W = {0}$.
  • Proof:
    Suppose $v \in V\cap W$. Then $v\in V$ and $v\in W$.
    We have $v^Tv=0=||v||^2$ which implies $v=0$

Property 3

  • Claim:
    For any vector $x\in R^n$,
    we can have $x=x_r+x_n$
    Then $x_r \in C(A^T)$ and $x_n \in N(A)$

  • Proof:
    這邊要證明$x_r$跟$x_n$加在一起可以湊出$R^n$的向量空間。
    我們已知$C(A^T)$跟$N(A)$的rank剛好是r 及n-r。
    所以只要從他們兩個空間中分別挑出r以及n-r兩組向量，然後他們又彼此獨立的話，
    就能夠湊齊n個獨立向量，這樣就能展開$R^n$的向量空間
    (前面的property 1有提到n個獨立的向量就能展開$R^n$的向量空間)
    Let $v_1, v_2, … v_r$ be a basis for $C(A^T)$ and $w_1, w_2, … w_{n-r}$ be a basis for $N(A)$
    To proof $v$ and $w$ are linear independent, we suppose $a_1v_1+a_2v_2+… a_rv_r+b_1w_1+b_2w_2+b_{n-r}w_{n-r} = 0$
    Let $u = a_1v_1+a_2v_2+… a_rv_r = -(b_1w_1+b_2w_2+b_{n-r}w_{n-r})$
    Then $u\in C(A^T)$ and $u\in N(A)$
    Since $C(A^T)\perp N(A)$, we must have $u=0$
    Since $v_1, v_2, … v_r$ are a basis for $C(A^T)$ and $w_1, w_2, … w_{n-r}$ are a basis for $N(A)$
    (因為v跟w剛好是從$C(A^T)$還有$N(A)$挑出來的basis，他們一定線性獨立，
    只有當係數都是零的時候，線性組合才會是零)
    We have $a_1=a_2=…=b_1=b_2 = 0$
    Therefore $v_1, v_2, … v_r$ and $w_1, w_2, … w_{n-r}$ are independent and form a basis for $R^n$
    (我們得到v跟w線性獨立，這就滿足了n個線性獨立的向量，
    可以展開$R^n$的目標，所以我們證明了$R^n$的任一個向量，
    可以透過這樣的分解方式得到)
    For every $x\in R^n$, we can have

    $$ x=\underline{c_1v_1+c_2v_2+… +c_rv_r}\ +\ \underline{c_{r+1}w_1+c_{r+2}w_2+…+c_nw_{n-r}} =x_r + x_n$$
    where $x_r \in C(A^T)$ and $x_n\in N(A)$

  • Remark:
    前面證明這個decomposition必定存在
    因為v跟w都是basis，所以他們要表達成的$x_r$及$x_n$係數組合是唯一的，
    也因此這樣的分解方式會是唯一的
    The decomposition $x=x_r+x_n$ is unique.
  • Remark:
    For every $x\in R^n$, $x=x_c+x_{ln}$
    where $x_c\in C(A)$ and $x_{ln} \in N(A^T)$.

Property 4

  • Claim:
    $Ax=A(x_r+x_n)=Ax_r + 0=Ax_r$
    Via A,
    $x_n$ goes to 0: $Ax_n=0$
    $x_r$ goes to the column space: $Ax_r=Ax$

Orthogonality

Dot product of inner product (over $R^N$ )

$$v \cdot w = v^Tw = v1w1 + v2w2+v3w3+…$$
$$v \cdot v = v_1^2 + v_2^2 + …$$

Def: Orthogonality

Two vectors $\underline{v}$ and $\underline{w}$ are orthogonal if $\underline{v} \cdot\underline{w} = 0$

  • Remark:

    $v’w = 0$ <=> $|v+w|^2 = |v|^2 + |w|^2$

  • Proof:

    $|v+w|^2 = (v+w)’(v+w) = |v|^2 + |w|^2 + (v’w + w’v) = |v|^2 + |w|^2$

Def: Orthogonal Subspace

Two subspace V and W are orthogonal if $v^Tw = 0$ for all $v \in V$ and all $w \in W$

For Nullspace:

$$A\underline{x} = \begin{bmatrix}row\ 1 \newline row\ 2 \newline . \newline . \newline
row\ m\end{bmatrix}
\begin{bmatrix} \underline{x} \end{bmatrix}
=
\begin{bmatrix}0\end{bmatrix}$$

  • Every row of $A$ is orthogonal to $\underline{x}$
  • Any linear combination of rows in $A$ is orthogonal to $\underline{x}$
  • The nullspace $N(A)$ and the row space $C(A^T)$ are orthogonal subspaces of $R^N$

Similarly:

$$A^Ty = \begin{bmatrix}col\ 1\newline col\ 2\newline .\newline .\newline col\ m\end{bmatrix}\begin{bmatrix}\underline{y}\end{bmatrix}=\begin{bmatrix}0\end{bmatrix}$$

  • The left nullspace $N(A^T)$ and column space $C(A)$ are orthogonal subspaces of $R^M$

Summary

  • $N(A)\perp C(A^T)$
  • $N(A^T)\perp C(A)$

Def: The orthogonal complement

The orthogonal complement $V^\perp$ of a subspace $V$ contains every vector that is orthogonal to $V$, i.e.,
$$V^\perp = {w: \underline{w^T}\underline{v}=0, \forall \underline{v}\in V}$$

  • Remark:

    $v^\perp$ is also a subspace

  • Claim:

    • $(C(A^T))^\perp=N(A)$
    • 跟所有的row orthogonal 的subspace 就是null space

  • Claim:

    • $(N(A))^\perp=C(A^T)$

    • Proof:
      We need to showthat if $v^Tw=0,\ \forall \underline{w}\in N(A)$, then $\underline{v} \in C(A^T)$
      If $v\notin C(A^T)$, then we can add $v$ as an extra row to $A$

      $$A’=\begin{bmatrix}A\newline v’\end{bmatrix}\underline{x} = \begin{bmatrix}0\newline 0\end{bmatrix}$$
      It then follows $N(A’) = N(A)$ but $dim(C(A^T)) = r$ and $dim(C(A’^T))= r+1$
      while $dim(N(A’)) = dim(N(A))=n-r$ which yields a contradiction
